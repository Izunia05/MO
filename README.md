Harmonogramowanie zadań - Programowane Dynamiczne, Podział  iograniczenia i Heurystyka zachłanna
Student powinien opracować system harmonogramowania zadań o określonychczasach wykonania, kosztach opóźnienia i priorytetach, wykorzystując trzy wyżej wspomniane podejścia. 
Program powinien generować harmonogramy oraz wizualizować je w formie wykresów Gantta.
Student powinien przeprowadzić badania na zestawach zadań o różnej skali, analizując jakosć rozwiązań oraz czas wykonania każdej metody.
W sprawozdaniu powinny znaleźć się wnioski dotyczące kompromisu pomiędzy szybkością obliczeń a jakością wyników.
